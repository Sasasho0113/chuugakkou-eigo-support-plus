## サービス概要

### サービスの目的

「学校の授業では習ったのに、テストで点が取れない」

そのような悩みを抱える中学生は、少なくありません。

当サービスは、このギャップを埋めるために開発されました。

- 学校の教科書に対応した「わかりやすい解説」
- テスト出題形式（空欄補充、並び替え、英作文）に基づいた「実践的な問題」
- わからないことを「すぐに質問できる」環境

以上を通じて、「テストで点が取れる英語」を実現します。

### 対象学年

現在、中学1年生向けのコンテンツを提供しております。

中学2年生向けのコンテンツは、2024年6月リリース予定です。

## サービスの特徴

### 1. テスト形式に基づいた問題

学校の定期テストに出題される形式（空欄補充、並び替え、英作文など）で、問題が設計されています。

例えば、「三単現のs」という単元では：

```
【学校での教え方】
「He / She / It の時は、動詞に『s』をつける」

【当サービスでの学び】
1. 解説でルール理解
2. 基礎問題（選択肢問題）で確認
3. 応用問題（並び替え、英作文）で実践
4. 小テスト（定期テスト形式）で実力確認
```

このように段階的に学ぶことで、テスト本番で確実に点が取れるようになります。

### 2. わかりやすい解説

各単元の解説は、以下の構成になっています：

- **学習ゴール：** この単元で何ができるようになるか
- **基本ルール：** 文法の基本を図解と例文で説明
- **用途別例文：** 肯定文・否定文・疑問文など、用途ごとの使い方
- **よくある間違い：** 多くの生徒がする誤りと、その理由

中学生にもわかりやすく、かつ実用的な内容になっています。

### 3. 質問機能

「学校の先生に聞きにくい」「親では説明できない」という時に、すぐに質問できます。

- **回答時間：** 24時間以内（通常は数時間以内）
- **回答者：** 英語教育の専門家
- **質問方法：** テンプレート選択で簡単入力

過去の質問と回答は、いつでも見直して復習に活用できます。

### 4. 学習記録の可視化

学習の進度が「見える化」されることで、モチベーションの維持が容易になります。

表示される情報：

- **連続学習ストリーク：** 「○日連続学習中」の表示
- **週間カレンダー：** 学習した日がチェック表示
- **単元別進捗：** 完了率がプログレスバーで表示
- **正答率トレンド：** 週単位での正答率の変化をグラフ化
- **問題タイプ別分析：** 得意な問題形式・苦手な問題形式を自動分析

### 5. 保護者との連携機能

保護者様は、以下のことができます：

- **学習状況の確認：** リアルタイムで進度を把握
- **メッセージ送信：** お子さんへの励ましメッセージを送信
- **質問内容の確認：** どんなテーマについて質問しているかを把握
- **弱点の発見：** 正答率が低い単元を自動検出

### 推奨学習スケジュール

**期間別の学習目安：**

| 期間 | 学習頻度 | 1回の時間 | 単元数 | 期待される成果 |
|------|--------|---------|------|-------------|

| 3～4週間 | 週3～4回 | 20分 | 2～3単元 | 応用問題に挑戦 |
| 1～2ヶ月 | 週4～5回 | 30分 | 4～6単元 | 学校テストで得点向上 |

### 保護者様ができるサポート

#### 1. 毎日の声掛け

```
「今日も頑張ってるね」
「○日連続すごい！」
「英語が得意になってきたね」
```

小さな声掛けが、継続の源となります。

#### 2. メッセージ機能の活用

専用の「メッセージ機能」から、お子さんへ励ましを送ることができます。

```
「今週も5日も勉強してるんだ。頑張ってるね」
「質問をいっぱいしてるから、深く理解してるんだと思う」
「テストに向けて、最後の追い込み頑張ろう」
```

### 3. 目標設定の面談

月1回程度、お子さんと目標について面談することをお勧めします。

```
「今月は何の単元を終わらせたい？」
「次のテストで、英語は何点取りたい？」
「一番得意な問題形式は？一番苦手なのは？」
```

目標を「他人事」ではなく、「自分事」にすることが重要です。

### 4. テスト対策の活用

定期テスト直前（2週間前）には、以下の活用をお勧めします：

```
1. 保護者画面から「正答率が低い単元」を確認
2. その単元を「もう一度実施」するよう勧める
3. 小テストを複数回受験させる
4. テスト前日は「復習」と「質問」に時間を使う
```

## よくあるご質問

### Q1. 本当に効果がありますか？

**A. 個人差がありますが、正しく活用いただければ、確実に効果が出ます。**

重要な要件：

- **週3回以上の学習：** 週1～2回では効果が薄い
- **解説をしっかり読む：** 問題を解くだけではなく、解説を理解
- **継続が最も重要：** 1～2ヶ月の継続で、学校テストでの点数向上を期待できます

### Q2. 塾と併用できますか？

**A. はい、むしろ塾との併用をお勧めします。**

最適な組み合わせ：

- **塾での授業：** 新しい内容を習う
- **当アプリ：** 自宅での復習・反復練習・質問対応
- **学校の授業：** 総合的な理解確認

### Q3. 学校の先生の教え方と違う場合は？

**A. 学校の先生の教え方を優先してください。**

当サービスは、一般的な英文法に基づいています。学校によっては、独自の教え方をされている場合があります。

「学校ではこう習ったのに、アプリと違う」という場合は、質問機能で「学校での教え方を確認したい」と質問いただければ、対応いたします。

### Q4. 兄弟姉妹で利用できますか？

**A. はい、ファミリープランをご利用ください。**

ファミリープラン（月額1,480円）なら、2人目のお子さんまで追加できます。

各お子さんのログインID・学習記録は別々に管理されるため、プライバシーも保たれます。

### Q5. データセキュリティは大丈夫ですか？

**A. はい、最高レベルのセキュリティで保護しています。**

実装内容：

- **SSL/TLS暗号化：** 通信はすべて暗号化
- **日本国内サーバー：** 海外への情報流出の心配なし
- **毎日自動バックアップ：** データ消失の防止
- **個人情報の最小化：** 名前とメールアドレスのみ保有（住所・電話番号は不要）

詳細は「プライバシーポリシー」をご確認ください。

### Q6. キャンセル・解約はいつでもできますか？

**A. はい、いつでも解約できます。**

解約方法：

1. メニュー → プラン変更
2. 「解約する」を選択
3. 来月から課金が停止

解約手数料・違約金は一切かかりません。

## トラブルシューティング

### アプリが落ちる

**対応方法：**
1. アプリを完全に終了（バックグラウンド削除）
2. スマートフォンを再起動
3. Wi-Fi 環境で再度起動

改善しない場合は、お問い合わせフォームからご連絡ください。

### ログインできない

**対応方法：**
1. メールアドレスとパスワードを確認
2. Caps Lock が ON になっていないか確認
3. 「パスワードを忘れた」からリセット

### 質問が反映されない

**対応方法：**
1. インターネット接続を確認
2. [送信]ボタンを確認（グレーアウト状態なら送信不可）
3. キャッシュをクリア（設定 → アプリ管理）

### 学習記録が保存されない

**対応方法：**
1. ログアウト → ログインを実施
2. アプリを最新版にアップデート
3. ストレージ空き容量を確認（最低100MB推奨）

問題が解決しない場合は、サポートチームにご連絡ください。
