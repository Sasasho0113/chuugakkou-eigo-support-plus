### KPI（成功指標）

| 指標 | 目標値 | 計測方法 |
|---|---|---|
| モニター数 | 10名 | 登録数 |
| 継続率（2週間後） | 70%以上 | ログイン数 |
| 1問以上解いた率 | 90%以上 | 問題実施数 |
| 質問数 | 10件以上 | 質問フォーム |
| 継続意向（アンケート） | 70%「続けたい」 | アンケート |
| 満足度（5段階） | 4.0以上 | アンケート |

---

### リスクと対策

| リスク | 対策 |
|---|---|
| モニターが集まらない | 知人への直接声かけを優先、Zoomでの個別説明も実施 |
| 質問が多すぎる | 1日3件制限、FAQ自動返答の整備 |
| バグによる離脱 | 自分自身で全画面テスト実施 |
| フィードバックが少ない | 短いアンケートに絞る、回答者に特典提供 |
