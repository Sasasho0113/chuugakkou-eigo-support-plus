# サービス設計資料
## 中学校英語サポート＋

**作成日**: 2026年3月29日
**バージョン**: 1.0

---

## 1. サービス概要

### サービス名
**中学校英語サポート＋**

### キャッチコピー・コピーワーク

| 項目 | 内容 |
|------|------|
| **キャッチコピー** | 「わかる」より「点が取れる」英語へ。 |
| **サブコピー** | 学校英語を、テストで使える英語に変える。中1から始める、確実な英語学習。 |
| **ブランドトーン** | 「頼れる先輩」「わかりやすい先生」「安心感のある個別指導」 |
| **ミッション** | 英語が苦手な中学生が、定期テストで自信を持って点が取れるようになること |
| **ビジョン** | 日本全国の中学生が、英語学習に向き合える環境を作ること |

### ターゲットユーザー

#### Primary ユーザー（学生）
- **属性**: 中学1年生（特に英語が苦手、定期テストで点が取れない生徒）
- **特徴**:
  - 授業のペースについていけていない
  - 何がわからないかが曖昧
  - 先生や友達に質問しづらい傾向がある
  - スマートフォンを常に持ち歩いている
  - SNSやYouTubeなど動画配信に慣れている

#### Secondary ユーザー（保護者）
- **属性**: 30〜50代（お子様の学習進捗を把握したい層）
- **特徴**:
  - お子様の成績改善を強く望んでいる
  - 塾代の高さに課題感を持っている
  - スマートフォンで進捗確認できることに価値を感じる
  - 月額980円程度のサービス料金であれば検討可能

## 2. 課題設定

### 学生が直面する課題（ペイン）

| # | 課題 | 具体的な状況 | 影響度 |
|---|------|-----------|--------|
| 1 | **授業が速すぎてついていけない** | 先生の説明速度が速く、ノートを取る余裕がない。次の単元が始まったときに前単元が理解できていない状態になる。 | 高 |
| 2 | **何がわからないかわからない** | 漠然とした不安感があり、どこから勉強し直すべきか判断できない。 | 高 |
| 3 | **先生や友達に聞きづらい** | 「こんなことを聞いたら恥ずかしい」という心理的ハードルがある。授業後に質問する勇気がない。 | 高 |
| 4 | **教材が難しく、自習が進まない** | 参考書や問題集の説明が簡潔すぎて理解できない。わからないままページが進む。 | 高 |
| 5 | **テストが近づくと焦る** | 直前対策をしたくても、どこから手をつければいいか不明確。得点には結びつかない。 | 中 |
| 6 | **親も教えられない** | 親自身が英語に不安があり、子どもからの質問に答えられない。 | 中 |

### 保護者が直面する課題（ペイン）

| # | 課題 | 具体的な状況 | 影響度 |
|---|------|-----------|--------|
| 1 | **塾代が高い** | 週2〜3回の塾通学で月1〜2万円の負担がある。 | 高 |
| 2 | **お子様の学習状況が見えない** | 実際にどの単元で困っているのか、進捗がどうなのか把握できない。 | 高 |
| 3 | **時間的な拘束** | 塾の送迎時間が負担。子どもが塾に依存しすぎる傾向がある。 | 中 |

## 3. 価値提案（4つのValue）

### Value 1: 成績アップ（テストに直結）
- **何が違うのか**: 定期テストに直結した解説・問題のみを厳選
- **どう実現するか**: 現役・元教員が監修。各単元の「テストに出やすい部分」を強調
- **期待効果**: 利用開始後、初回定期テストで平均+15点

### Value 2: 質問しやすさ（心理的ハードルを下げる）
- **何が違うのか**: 匿名で気軽に質問できる環境
- **どう実現するか**:
  - ユーザー名を表示しない質問・回答機能
  - 質問テンプレートで質問しやすくする
  - 原則24時間以内の回答保証
- **期待効果**: 初月の質問数は利用者の30〜50%が利用

### Value 3: 価格の手頃さ（経済的アクセス）
- **何が違うのか**: 月額980〜1,480円の低価格
- **どう実現するか**: スケーラブルなWeb配信で運用コストを最小化
- **期待効果**: 塾代の1/15以下で利用可能

### Value 4: 教員視点（信頼できる説明）
- **何が違うのか**: 現役・元教員による監修
- **どう実現するか**: 各解説に「講師プロフィール」を表示
- **期待効果**: 学校教育との一貫性があり、保護者が安心できる

## 4. 競合との差別化

### 競合分析マトリックス

| サービス | 特徴 | 強み | 弱み | 本サービスの差別化 |
|---------|------|------|------|------------------|
| **スタディサプリ** | 動画配信型 | 著名講師、豊富な単元 | 受動的、質問機能なし | 演習・質問中心で主体的学習 |
| **学研 AI タブレット** | タブレット配信 | 対話型AI | 機械的、高額 | 人間による温かい回答 |
| **個別指導塾** | 対面型 | マンツーマン対応 | 高額（月2万円以上）、時間拘束 | Web完結、低価格、時間自由 |
| **YouTube** | 無料動画 | 無料、自由 | 体系性なし、質問機能なし | 単元別、テスト直結、質問対応 |
| **教科書準拠問題集** | 紙教材 | 実績あり | 時代遅れ、回答がない | 動的フィードバック、即時回答 |

### 本サービスのポジショニング

```
                    低価格 ←→ 高価格
                      ↑
                 本サービス
                   ★
         ┌─────────────────┐
         │   質問機能あり  │
         └─────────────────┘
    YouTube    スタディサプリ
        ★             ★

    ┌──────────────────────┐
    │  質問機能なし        │
    └──────────────────────┘
        ↓
    個別塾（高額・対面）
```

## 5. MVPの範囲

### MVP対象
- **学年**: 中学1年生のみ
- **単元**: 「三単現のs」（初期リリース）
- **学習期間**: 初月は1単元の完全実装に注力

### MVP機能一覧

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 1 | **ホーム画面** | 学習継続状況、次のステップ、今週のおすすめ単元を表示 | P1 |
| 2 | **単元一覧** | 利用可能な単元をカテゴリ別に表示 | P1 |
| 3 | **単元詳細（解説）** | 単元の基本解説とチェック問題 | P1 |
| 4 | **ワーク（演習）** | 10〜20問の演習問題と自動採点 | P1 |
| 5 | **質問機能** | ユーザーが質問を送信、スタッフが回答 | P1 |
| 6 | **質問履歴** | 過去の質問・回答を表示 | P2 |
| 7 | **学習記録** | 学習進捗（日数、時間、正答率）を可視化 | P1 |
| 8 | **保護者レポート** | 保護者向けの学習状況ダッシュボード | P2 |
| 9 | **LP（ランディングページ）** | サービス紹介、会員登録導線 | P1 |
| 10 | **会員管理** | ユーザー登録、ログイン、プロフィール編集 | P1 |

### MVP非対象（今後の拡張）
- AIチャット質問機能
- 単語帳・フラッシュカード
- リスニング機能
- スピーキング評価
- 定期テスト予想問題（初期は提供しない）
- BtoB向け管理画面

## 6. 事業モデル

### 収益化戦略

#### フェーズ1: 初期段階（利用者0〜1,000名）
- **モデル**: フリーミアムモデル
- **施策**:
  - 「三単現のs」単元は無料公開
  - 他単元はプレミアム機能として月額980円
  - 質問機能は全プラン対応

#### フェーズ2: 拡大期（利用者1,000〜10,000名）
- **モデル**: 複数プラン設定
  - **ベーシック**: 月額980円（1学年全単元）
  - **プラス**: 月額1,480円（複数学年対応）
  - **テスト対策プラン**: 1単元500円（期間限定）

#### フェーズ3: 成熟期（利用者10,000名以上）
- **モデル**: BtoB展開
  - 学校向けライセンス（生徒数に応じた変動費）
  - 塾向け提携プラン（帯域料金）
  - 教員向けダッシュボード（進捗管理ツール）

## 8. 今後の拡張方針

### 単元拡張計画

#### 中学1年生（MVP期に完成）
- **春学期**: 三単現のs、be動詞、過去形（was/were）、進行形
- **秋学期**: 不定詞、動名詞、助動詞、比較級・最上級

#### 中学2年生（6ヶ月目以降）
- 受動態、現在完了、関係代名詞、接続詞

#### 中学3年生（12ヶ月目以降）
- 仮定法、複文、英作文対策

### 機能拡張計画

| フェーズ | 追加機能 | 時期 | 理由 |
|---------|--------|------|------|
| **Phase 1** | AIチャット質問（FAQ自動返答） | 9ヶ月目 | 質問対応スケーラビリティ向上 |
| **Phase 2** | 単語帳・スペリングテスト | 12ヶ月目 | 補助教材として利用価値向上 |
| **Phase 3** | リスニング問題 | 15ヶ月目 | 高学年対応に必須 |
| **Phase 4** | 英作文採点機能 | 18ヶ月目 | 中3対策として高い需要 |

### BtoB展開計画

#### 学校向けライセンス（12ヶ月目以降）
- **ターゲット**: 公立中学校（全国3,000校）
- **モデル**:
  - 基本料金：月額50,000円
  - 追加ライセンス：生徒1名 / 月50円
  - 学校向け管理画面（教員ダッシュボード）
- **期待**: 初年度20校、年額12M円

#### 塾向けホワイトラベル（15ヶ月目以降）
- **ターゲット**: 個別指導塾、オンライン塾
- **モデル**:
  - API連携で既存プラットフォームに統合
  - 月額：生徒50名あたり10,000円
- **期待**: 初年度50塾、年額5M円

## 10. リスク管理

### 主要リスク

| # | リスク | 発生確度 | 影響度 | 対策 |
|---|--------|--------|--------|------|

| 4 | **サーバーダウン** | 低 | 高 | マルチリージョン対応、監視ツール導入 |
